## 🚀 EXECUTION OPTIONS

### Option 1: Maven Commands
```bash

mvn clean test

# Run specific test
mvn test -Dtest=LoginTest

# Run specific method
mvn test -Dtest=LoginTest#testValidLogin
```

### Option 2: PowerShell Script
```bash
.\run-tests.ps1 -Action all          # All tests
.\run-tests.ps1 -Action login        # Login tests only
.\run-tests.ps1 -Action cart         # Cart tests only
.\run-tests.ps1 -Action logout       # Logout tests only
```
