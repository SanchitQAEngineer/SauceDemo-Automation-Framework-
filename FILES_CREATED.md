## 📁 Java Source Files (src/test/java)

### Base Classes (base/)
1. **BaseTest.java**
   - Parent class for all test classes
   - Handles WebDriver initialization and cleanup
   - Sets up Chrome/Edge browsers
   - Implements @BeforeMethod and @AfterMethod

### Page Object Classes (pages/)
2. **LoginPage.java**
   - Login page object model
   - Methods: login(), getErrorMessage(), isLoginPageLoaded()
   - Locators for login form elements
   - 8+ methods for login operations

3. **HomePage.java**
   - Home page object model
   - Methods: addProductToCart(), removeProductFromCart(), logout()
   - Product browsing and shopping cart operations
   - 10+ methods for home page interactions

4. **CartPage.java**
   - Shopping cart page object model
   - Methods: getCartItemCount(), removeItemByIndex(), isCartEmpty()
   - Cart item management
   - 9+ methods for cart operations

### Test Classes (tests/)
5. **LoginTest.java**
   - 5 test methods for login functionality
   - Includes: valid login, invalid login (data-driven), locked user, empty fields
   - Uses @DataProvider for Excel-based test data
   - Comprehensive login scenario coverage

6. **CartTest.java**
   - 7 test methods for shopping cart functionality
   - Includes: add/remove products, view cart, clear cart
   - Tests cart operations on both home and cart pages
   - Product verification tests

7. **LogoutTest.java**
   - 4 test methods for logout functionality
   - Tests: basic logout, logout after shopping, re-login
   - Session and cart behavior validation

### Listener Classes (listeners/)
8. **TestListener.java**
   - Implements ITestListener for TestNG events
   - Integrates Extent Reports
   - Captures screenshots on test failure
   - Provides test execution logging

### Utility Classes (utils/)
9. **ExcelReader.java**
   - Reads test data from Excel files using Apache POI
   - Methods: getCellData(), getCellDataByHeader(), getRowDataAsMap()
   - Support for both row and column data retrieval
   - Flexibility for different data formats

10. **ScreenshotUtil.java**
    - Captures screenshots during test execution
    - Methods: takeScreenshot(), takeScreenshotWithCustomName()
    - Automatic screenshot organization
    - Directory management utilities

11. **ConfigReader.java**
    - Reads configuration from properties file
    - Methods: getProperty(), getBrowser(), getBaseUrl(), getExplicitWait()
    - Centralized configuration management
    - Default values for missing properties

12. **ExcelDataCreator.java**
    - Generates test data Excel files
    - Creates LoginTestData.xlsx with invalid login scenarios
    - One-time utility for test data setup
    - Can be extended for other test data files

## 🚀 Getting Started

### 1. First Time Setup
```bash
# Run setup script
.\setup.bat

# Or manually:
mvn clean install
mvn exec:java -Dexec.mainClass="utils.ExcelDataCreator"
```

### 2. Run Tests
```bash
# Option 1: Using Maven
mvn clean test

# Option 2: Using PowerShell script
.\run-tests.ps1 -Action all
```

### 3. View Results
- Reports: `reports/ExtentReport_<timestamp>.html`
- Screenshots: `screenshots/` folder
